# Data & Software

The Shepherd Research Lab makes research resources — data and software — available to the scientific community. Some tools are freely downloadable; others require an approval process or portal login.

</div>
</section>

## Data

Researchers can request data through the Analysis Plan Form. Additional data resources are hosted on specialized platforms:

- **Shape Up!** — data-access portal (login credentials required)
- **HIPIMR — Breast Health Questionnaire**
